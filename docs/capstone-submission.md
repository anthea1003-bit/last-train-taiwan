## Kaggle Writeup Draft

Title:

`The Last Impossible Round-Island Train: A Free-Tier Taiwan Travel Agent Game`

Suggested structure:

1. **Problem / Idea**: make the 5-Day AI Agents concepts tangible through a
   public, free, bilingual adventure game rather than a paid cloud demo.
2. **What I Built**: describe the six Taiwan landmarks, hidden Penghu seventh
   station, memory/stamp system, and local Conductor Agent.
3. **Agentic Workflow**: explain Codex collaboration plus Antigravity
   refinement, Agent Skills, and development-time MCP research.
4. **Safety and Cost Boundary**: 2-tier fallback — Gemini 2.5 Flash via a Cloudflare Worker API proxy that securely stores and rotates three Free Tier API keys (no billing enabled). If the Worker fails, the game automatically falls back to local rules. Zero credential exposure in frontend, no cost, no personal data, no server database.
5. **Evaluation**: summarize unit tests, localization integrity checks,
   non-spoiler Agent tests, mobile/desktop verification, and hidden-ending
   checks.
6. **What I Learned**: vibe coding becomes production-grade when constrained
   by specs, tests, security rules, and a clear demo path.
7. **ADK Architecture Choice**: We made a deliberate decision to bypass cloud-hosted Vertex AI or Google ADK. Instead of building a server-heavy workflow, we implemented the 2-tier fallback and tool execution loop in a browser-native (TypeScript) architecture, using a lightweight serverless Cloudflare Worker solely as a secure proxy to inject and rotate API keys. This proves that sophisticated Agentic workflows can run entirely serverless, zero-cost, private, and secure.
